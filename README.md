# AI-Health-News-Agent
AI-Health-News-Agent 是一个基于 Node.js 的 AI 代理程序，每日获取公共卫生领域的最新热点新闻，生成 Markdown/HTML 报告，并通过邮件发送。

## 项目目录结构


## 📌 功能
- 爬取 WHO、CDC、Nature 等公共卫生新闻
- 生成 Markdown 和 HTML 报告
- 通过 API 提供新闻数据
- 每天 8:00 自动执行并发送邮件
